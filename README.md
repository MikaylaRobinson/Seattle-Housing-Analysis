# Seattle-Housing-Analysis
Data analysis for housing prices in King County, WA. 

## Data Overview
The data is a collection of housing prices as well as many other characteristics, bedroom and bathroom count, location, size. 

## Tools
* Python 
* Pandas
* Seaborn
* Scikit-learn
* Jupyter
